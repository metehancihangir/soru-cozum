# Icon System — Medium Design System

## Detection Result

**No standard icon library was detected** on medium.com.

Medium uses **custom inline SVG icons** — proprietary, not from a public library.

## Common Icon Libraries to Consider as Substitutes

If you are building a Medium-inspired interface and need icons, the following libraries
match Medium's minimal aesthetic well:

| Library | Style | License |
|---------|-------|---------|
| [Heroicons](https://heroicons.com) | Minimal line/solid | MIT |
| [Phosphor Icons](https://phosphoricons.com) | Clean, scalable | MIT |
| [Feather Icons](https://feathericons.com) | Ultra-minimal stroke | MIT |
| [Lucide](https://lucide.dev) | Feather fork, maintained | MIT |

## Usage Guidance

Given Medium's design philosophy (minimal, content-first):
- Use **stroke/outline** icon variants, not filled
- Size: `20px` for UI icons, `16px` for inline
- Color: `var(--color-neutral-dark)` (`#242424`) for primary icons, `var(--color-neutral-mid)` (`#6b6b6b`) for secondary
- No color icons — monochrome only
