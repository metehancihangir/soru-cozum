## Design Philosophy (TL;DR)

Medium's design system is built on **content-first minimalism**:

- **No color** — only black, white, grey. Brand identity is carried by typography, not color.
- **Typography = Identity** — GT Super (serif) for display, Sohne (sans-serif) for UI.
- **Pill buttons** — `border-radius: 9999px` equivalent on all buttons.
- **Whitespace hierarchy** — depth and hierarchy via spacing, not shadows.
- **Instant transitions** — `transition: 0.001s` (effectively no animation).
